O que é HTML, CSS e JavaScript?

HTML

HTML é a estrutura do conteúdo — marcação semântica para texto, imagens e links.

<h1>Título</h1>

Quando usar: estruturar conteúdo e semântica.

CSS

CSS é a linguagem de estilos que define aparência, layout e responsividade.

h1 { color: #2563EB; }

Quando usar: estilizar e tornar responsivo.

JavaScript

JS adiciona comportamento: interações, DOM e persistência no cliente.

document.querySelector('h1').textContent = 'Olá!';

Quando usar: interações e lógica dinâmica.
