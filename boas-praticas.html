Boas práticas profissionais

  • Use landmarks (header, main, footer).
  • Forneça texto alternativo descritivo em imagens.
  • Conecte labels a inputs (for/id).
<label for="nome">Nome</label><input id="nome"/>

  • Mobile-first com media queries.
  • Use Flexbox e Grid para layouts adaptativos.
  • Prefira unidades relativas (rem, %) para tipografia.
@media (min-width: 600px) { .grid { grid-template-columns: repeat(2, 1fr); } }

  • Use BEM ou outra convenção para nomes.
  • Use variáveis CSS para cores e tipografia.
  • Comente e separe layouts de componentes.
.
.btn--primary { background: var(--color-primary); }
